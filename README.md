## 📌 Overview

**Language_Detection_Model AI** is a machine learning-based system that detects the language of any given text input in real time.

💬 Example:
- "Hello, how are you?" → English 🇺🇸  
- "Bonjour tout le monde" → French 🇫🇷  
- "Hola amigo" → Spanish 🇪🇸  
- "नमस्ते" → Hindi 🇮🇳
